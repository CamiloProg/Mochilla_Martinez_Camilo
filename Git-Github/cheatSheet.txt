git init
Sintaxis: git init
Descripción: Inicializa un nuevo repositorio Git en el directorio actual.

git clone
Sintaxis: git clone <URL del repositorio>
Descripción: Clona un repositorio existente en un nuevo directorio.

git add
Sintaxis: git add <archivo>
Descripción: Añade un archivo al área de preparación (staging area) para ser commitado.

git commit
Sintaxis: git commit -m "mensaje del commit"
Descripción: Crea un nuevo commit con los archivos en el área de preparación y les asigna un mensaje.

git push
Sintaxis: git push <nombre_remoto> <nombre_rama>
Descripción: Envía los commits locales a un repositorio remoto.

git pull
Sintaxis: git pull <nombre_remoto> <nombre_rama>
Descripción: Descarga los cambios del repositorio remoto y los fusiona con la rama local actual.

git status
Sintaxis: git status
Descripción: Muestra el estado actual del repositorio, incluyendo los archivos modificados, añadidos y eliminados.

git branch
Sintaxis: git branch
Descripción: Muestra una lista de las ramas del repositorio y resalta la rama actual.

git checkout
Sintaxis: git checkout <nombre_rama>
Descripción: Cambia a la rama especificada y actualiza el directorio de trabajo al último commit de dicha rama.

git merge
Sintaxis: git merge <nombre_rama>
Descripción: Fusiona la rama especificada con la rama actual.

git log
Sintaxis: git log
Descripción: Muestra un historial de commits con información detallada, incluyendo el autor, la fecha y el mensaje.

git remote
Sintaxis: git remote add <nombre_remoto> <URL_remoto>
Descripción: Agrega un nuevo repositorio remoto con el nombre especificado y la URL proporcionada.
